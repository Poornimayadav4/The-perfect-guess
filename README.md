# Perfect Guessing Game 🎯

A simple number guessing game built using Python.

## Features
- Generates a random number.
- Takes user input to guess the number.
- Provides feedback if the guess is too high or too low.
- Counts the number of attempts taken by the player.
- Displays a success message when the correct number is guessed.

## Technologies Used
- Python
- Random Module
- Loops
- Conditional Statements

## Learning Outcomes
- Working with random numbers
- User input handling
- Loop implementation
- Decision-making using if-else statements

## How to Run
1. Open the project in Python.
2. Run the program.
3. Enter your guesses until you find the correct number.
